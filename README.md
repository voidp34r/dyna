# Usage
<!-- usage -->
```sh-session
$ npm install -g dyna
$ dyna COMMAND
running command...
$ dyna (-v|--version|version)
dyna/0.0.0 linux-x64 node-v10.15.2
$ dyna --help [COMMAND]
USAGE
  $ dyna COMMAND
...
```
